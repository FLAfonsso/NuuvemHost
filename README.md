# ☁️ Nuuvem Host

Projeto fictício de uma empresa de hospedagem de sites, desenvolvido com **HTML5**, **CSS3** e reatividade. O site simula uma plataforma de serviços de hospedagem, com planos, busca de domínio, área de contato e informações sobre segurança, performance e suporte.

## 🚀 Funcionalidades

- Página inicial com destaques: **Segurança**, **Performance** e **Suporte 24/7**
- Seção **Planos e Preços** com cards interativos:
  - Básico (R$15/mês)
  - Dedicado (R$30/mês)
  - Dedicado Plus (R$50/mês)
  - Cloud (R$100/mês)
- Verificador de **domínio disponível** (front-end)
- Layout responsivo e navegação fluida

## 🛠️ Tecnologias utilizadas

- HTML5 semântico
- CSS3 (Flexbox/Grid, responsividade)
